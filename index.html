Computer Vision: Brain Tumor Detection using Deep Learning Predictive Models in Python

In a collaborative group project involving three members, we focused on applying deep learning models such as Deep Neural Networks (DNNs), Convolutional Neural Networks (CNNs), and Artificial Neural Networks (ANNs) for image classification, with a potential application in medical diagnostics for brain tumors. Our approach began with defining the problem statement: enhancing image classification accuracy, especially in medical imaging contexts. We utilized a labeled dataset with "yes" and "no" classifications to train our models effectively. The project involved extensive exploratory data analysis (EDA), data preprocessing, and augmentation techniques to prepare the dataset. We constructed various models, including those utilizing transfer learning with VGG16, to optimize classification performance. Model evaluation focused on accuracy, precision, and computational efficiency. The project aims to improve the classification of brain MRI images for earlier and more accurate diagnoses, contributing valuable insights into the strengths and limitations of different neural network architectures.

Student Academic Performance Analysis using R

This project scrutinized student academic performance by analyzing a dataset with attributes such as gender, school, family background, and alcohol consumption. The objective was to uncover how these factors relate to academic success. Key features of the project include Exploratory Data Analysis (EDA) involving visualization of data to identify patterns as well as statistical analyses to explore various factors like gender differences in grades, the impact of different schools on performance, the effect of alcohol consumption on final grades, the influence of romantic relationships on grades and attendance, and the role of travel time in academic success. Tools used include R and RStudio for data analysis, with packages like tidyverse for data manipulation and gridExtra for advanced plotting. Results were communicated through effective visualizations such as box and violin plots.

Car Rental System using Python Programming

The Super Car Rental Services (SCRS) project is a Python-based program developed for an assignment to streamline the car rental process. It features distinct functionalities for three user roles: Admin, Registered User, and Non-Registered User.
